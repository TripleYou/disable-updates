# Disable Updates
> Disable WordPress update functionality

## Features
- WordPress Core Updates
- WordPress.org Plugin Updates
- WordPress.org Theme Updates
- 3rd party (non-WP.org) Plugins
  - Gravity Forms
  - Gravity Forms Add-Ons
